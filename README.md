# Piyush Kumar Singh - Personal Portfolio & Research Blog

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

Welcome to the source code of my personal website and blog! This project serves as a centralized hub for my physics research, mathematics notes, programming experiments, and personal portfolio. 

## 🚀 Features

This project was built with modern web technologies, prioritizing both stunning aesthetics and blazing-fast performance:

- **Astro & Static Site Generation:** Built on the Astro framework to deliver zero-JS static HTML by default, ensuring lightning-fast load times.
- **Content Collections:** Type-safe management of Markdown (`.md`) files for both Blog Posts and Projects.
- **Advanced Math Rendering:** Full KaTeX integration allows for seamless rendering of complex Physics and Mathematics equations directly within Markdown.
- **View Transitions:** Hardware-accelerated, native browser view transitions for buttery-smooth page navigations without full page reloads.
- **High-Performance Animations:** GPU-accelerated reveal-on-scroll animations and interactive UI components.
- **Beautiful Typography & Code:** Integrated with Tailwind Typography and Expressive Code to deliver a premium reading experience and gorgeous syntax highlighting.
- **Responsive Dark Theme:** A sleek, fully custom dark theme utilizing glassmorphism, dynamic grids, and modern UI/UX design paradigms.
- **Automated Deployments:** Fully optimized for seamless Continuous Deployment via Netlify.

## 🛠️ Tech Stack

- [Astro 4](https://astro.build/) - The core web framework.
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework for rapid UI development.
- [KaTeX](https://katex.org/) - The fastest math typesetting library for the web.
- [Expressive Code](https://expressive-code.com/) - Engine for beautiful code blocks.
- [Lucide Icons](https://lucide.dev/) - Beautiful and consistent iconography.

## 🏃‍♂️ Running Locally

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Start the Development Server:**
   ```bash
   npm run dev
   ```
   *Note: As per our internal workspace rules, you can also run `astro dev --background` to run the server in the background.*

3. **Build for Production:**
   ```bash
   npm run build
   ```
