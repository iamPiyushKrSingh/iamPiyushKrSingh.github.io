## Mathematics

I have not taken many mathematics courses as compared to a math major student, but I have been taking a few courses in the past few years.

- **Autumn 2024**
  - MA3102 Algebra I\
    In this course, we studied the group theory. For this course, I have my handwritten class notes, which can be found [here](https://gitlab.com/latex-notes-iiserk/ma3102/-/raw/main/notes/notes.pdf).
  - MA3105 Numerical Analysis\
    In the course, we studied root finding methods, numerical calculus, solving linear systems, and eigenvalue problems. All the code and assignments for this course are available [here](https://gitlab.com/latex-notes-iiserk/ma3105/-/tree/main/).

    For our end-semester project, we studied `power method` for finding the eigenvalues of a matrix. I have completed this project along with my group mates [Gautum Singh](https://www.linkedin.com/in/gautam-singh-81466a218/), and [Sabarno Saha](https://theinvisiblefoe.github.io/). The report is available [here](https://gitlab.com/latex-notes-iiserk/ma3105-project/-/raw/main/report/report.pdf), final presentation slides are available [here](https://gitlab.com/latex-notes-iiserk/ma3105-project/-/raw/main/ppt/ppt.pdf), and all the codes are available [here](https://gitlab.com/latex-notes-iiserk/ma3105-project/-/tree/main/codes).

- **Spring 2024**
  - MA2201 Analysis II\
    Taught by [Dr. Rajib Dutta](https://https://math.iiserkol.ac.in/faculties/rajib.html), this course covered the topics of differentiability, integration, and sequences and series of functions. He provided handwritten notes for this course, which can be found [here](https://gitlab.com/latex-notes-iiserk/ma2201/-/tree/main/notes).
  - MA2202 Probability I\
    Taught by [Dr. Soumya Bhattacharya](https://math.iiserkol.ac.in/faculties/soumya.html), this course covered the topics of axiomatic probability, conditional probability, Bayes' theorem, Random variables, memorylessness, law of large numbers, (discrete time) Markov chains, the central limit theorem, and order statistics. He provided [typed notes](https://typst.app/) for this course, which can be found [here](https://gitlab.com/latex-notes-iiserk/ma2202/-/raw/main/notes/lecture_notes.pdf).

    I have my solutions to all the assignments of this course, which can be found [here](https://gitlab.com/latex-notes-iiserk/ma2202/-/tree/main/assignments).
